# ⛓️ Ephemeral Protocol

> **The Battery-Safe Mobile Blockchain**

Ephemeral Protocol is a new Layer-1 blockchain built in **Rust**. It is designed to let everyday smartphones mine crypto and verify blocks directly inside a mobile browser—**with 0% noticeable battery drain or phone heating**.

## 🚀 How It Works (The Simple Way)

Traditional crypto mining requires massive, power-hungry computers. Ephemeral fixes this by splitting the work:

* **The Heavy Lifters:** Powerful computers bundle transactions and create tiny mathematical proofs.
* **The Mobile Nodes (You):** Your phone simply downloads a tiny proof and checks it using high-speed WebAssembly. This check takes less than **10 milliseconds**, letting your phone earn rewards while its CPU stays asleep 99% of the time.

## 🛠️ The Tech Stack

* **Core Engine:** Rust (For ultra-fast, secure cryptographic checks)
* **Frontend:** HTML5 / CSS3 / JavaScript (Custom dashboard telemetry)
* **Execution:** WebAssembly (To run native Rust code at full speed inside mobile browsers)
